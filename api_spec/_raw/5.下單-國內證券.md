GetAvgCost
| 查詢昨日未沖銷的證券明細。 |
|---|
| 宣告 | Long GetAvgCost ([in] BSTR bstrLogInID, [in] struct SKAVGCOST pOrder,[out] BSTR* bstrMessage); |
| 參數 | bstrLogInID | 登入ID。 |
